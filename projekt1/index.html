TI Projekt 1

Zawartość merytoryczna projektu

Projekt przedstawia prostą symulacje ruchu rzuconej piłki w 2D przy zadanych warunkach początkowych.

Całość znajduje się w zakładce "Projekt".

Język HTML5

W ramach projektu wykorzystano elementy semantyczne języka HTML5 tj. <header>,<nav> czy <article>. Treści multimedialne prezentowane są z wykorzystaniem nowych elementów tj. <audio>.

Grafika w projekcie

W ramach projektu opracowano grafikę z wykorzystaniem elementu <canvas>.

JavaScript w projekcie

W ramach projektu wykorzystano język JavaScript do modyfikacji styli CSS oraz do modyfikacji dokumentu HTML5 z wykorzystaniem funkcji "getElementByID" oraz "innerHTML". Do obsługi zdarzeń wykorzystano wszelakie funkcje on..().

Walidacja projektu

Strony poprawnie walidują się na stronie W3C zgodnie z standardem HTML5 i CSS 3.0.

Ze względów kompatybilności w celu obejrzenia projektu uprasza się o korzystanie z przeglądarki FireFox.
